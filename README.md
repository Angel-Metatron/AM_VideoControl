# Управление просмотром видео (AM_Video_Control)
Добавляет управление с помощью горячих клавиш режимом просмотра видео в Youtube и плеерах других сайтов (PlayerJS, например).

Сейчас в ютубе много ИИ-видео в книжном формате с полями по бокам типа шортсов, или для шортсов открытых через `/watch?v=id`, с этим скриптом видео можно повернуть и растянуть видео на весь экран для экранов в книжной ориентации. Отразить видео если хочется посмотреть его вверх-ногами :) Или если хочется рассмотреть чего на видео увеличив это на весь экран.

Скрипт для [TamperMonkey](http://tampermonkey.net/) и ему подобных.

### Возможности:
- Поворот на 90 грд (клавиша R (англ) или К (русск)) по часовой стрелке.
- Отражение по горизонтали (клавиша H (англ) или Р (русск)), по вертикали (клавиша V (англ) или М (русск)).
- Масштаб (Alt + Колесо мыши, Alt + Num-/+) от -10x до +10x, сброс масштаба (Alt + 0/Num0). Перемещение по увеличенному видео с помощью перетаскивания мышкой.
- Запоминание настроек при переходе из обычного режима в полноэкранный и наоборот.
- Настройка клавиш пока только правкой самого скрипта в блоке настроек. (не заморачивался)
